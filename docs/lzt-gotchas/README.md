# Грабли API lzt.market / lolzteam

Здесь — только то, что узнали о **чужом сервере**: что он возвращает, чего
не возвращает, как врёт, как медленно отвечает, что отклоняет. Ничего про
наш код, наши обёртки или наши решения — только про их API.

Каждый файл — своя тема:

- [`timing.md`](timing.md) — медленные эндпоинты, таймауты, лимиты запросов,
  что значит `retry_request`.
- [`responses.md`](responses.md) — что API пропускает, шлёт `null`-ом или
  типизирует не так, как заявляет спецификация.
- [`catalog.md`](catalog.md) — категории, слаги, поиск, размер и порядок
  параметров фильтров.
- [`auth.md`](auth.md) — токены, ключи, что на самом деле значат коды
  ошибок авторизации.
- [`events.md`](events.md) — почему на маркете нет событий и что с этим
  делать.

Источник каждой находки — путь в этом репозитории (модели, фасады SDK,
адаптер маркета, движок событий). Даты в тексте — когда конкретное
расхождение было поймано на живом сервере.
