Выкладываю клиент, который мы пилили полгода. Типизированный, асинхронный, с нормальными ошибками вместо except Exception: pass.

Главное отличие от закрытых сборок — можно прочитать, что он делает с твоим токеном.

pip install lzt-sdk from lzt import Client async with Client(token) as c: items = await c.market.search(price_max=500)

Дальше — планировщик задач и плагины. Монолит переписывать не буду, вместо этого выношу движок, чтобы следующие тулзы строились поверх, а не с нуля.

  • market.search / market.buy / market.check
  • profile.me / profile.balance
  • threads.bump / threads.create
